# Canvas/Node.js Pong

### Usage

$ npm install websocket  
$ node pongServer  
$ index.html  
...  
// Player two.  
$ index.html  

_Note_: This example uses port 1337. If you're not running the example locally, you will need to modify the target hostname in index.html. ( buildWebSock() )
